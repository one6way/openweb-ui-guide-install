# Документация по установке и настройке Open WebUI

## Структура документации

1. **01_INSTALLATION_PYTHON.md** - Установка Open WebUI как Python приложения (без Docker)
2. **02_INSTALLATION_DOCKER.md** - Установка Open WebUI с помощью Docker
3. **03_ADDING_MODELS.md** - Добавление языковых моделей
4. **04_ADVANCED_CONFIGURATION.md** - Расширенная конфигурация (LDAP, Keycloak, логирование)
5. **05_DNS_CONFIGURATION.md** - Настройка DNS

## Быстрый старт

### Выбор метода установки

- **Python установка** - если Docker недоступен или нежелателен
- **Docker установка** - если Docker уже установлен или предпочтительнее

### Последовательность действий

1. Выберите метод установки (Python или Docker)
2. Следуйте соответствующей инструкции
3. После установки добавьте модели (см. 03_ADDING_MODELS.md)
4. Настройте аутентификацию и логирование (см. 04_ADVANCED_CONFIGURATION.md)
5. Настройте DNS (см. 05_DNS_CONFIGURATION.md)

## Требования

### Для Python установки:
- CentOS 9
- Python 3.11
- Минимум 2GB свободного места
- Права root или sudo

### Для Docker установки:
- CentOS 9
- Docker установлен
- Минимум 3GB свободного места
- Права root или sudo

## Поддержка

При возникновении проблем:
1. Проверьте логи сервиса
2. Убедитесь что все зависимости установлены
3. Проверьте конфигурационные файлы
4. См. раздел "Решение проблем" в соответствующей инструкции
